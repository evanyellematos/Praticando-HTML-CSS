Aprendendo Front-End do Zer0!

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

CSS (Folhas de Estilo em Cascata) permite a você criar páginas web agradáveis. A web seria um lugar chato se todos os web sites tivessem estilos iguais. Usando CSS você pode controlar exatamente a aparência dos elementos HTML no navegador, apresentando a sua marcação com o design que desejar. É uma linguagem para especificar como documentos são apresentados aos usuários — como eles são estilizados, dispostos etc.
